📊 Retail Sales Performance & Customer Insights Dashboard
🚀 Project Snapshot

Built an interactive Power BI dashboard to analyze $2.26M in retail sales data, uncover growth opportunities, monitor business performance, and support smarter decision-making through data-driven insights.
This project transforms raw transactional data into a clear executive dashboard covering sales trends, customer behavior, category performance, top products, and regional demand.

📁 Project Overview

Retail businesses generate massive amounts of sales data every day. Without proper analysis, valuable opportunities remain hidden.
This dashboard was designed to convert raw retail data into actionable insights that help businesses answer critical questions such as:

Which categories generate the most revenue?
Which customer segments drive growth?
Which products perform best?
Which regions need more attention?
How are monthly sales changing over time?

🎯 Business Objectives

✔️ Track key KPIs in one centralized dashboard
✔️ Identify top-performing products and categories
✔️ Analyze customer segments and buying behavior
✔️ Compare sales performance across regions
✔️ Monitor monthly growth trends
✔️ Enable faster strategic decision-making
